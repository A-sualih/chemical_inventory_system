✅ 2. Secure Login (JWT / Session-Based)

👉 Recommended: JWT (JSON Web Token)

👉 What happens:

User logs in
Server checks password
Server sends token
Client stores token (localStorage or cookie)

👉 Backend

import jwt from "jsonwebtoken";

app.post("/login", async (req, res) => {
  const user = await User.findOne({ email: req.body.email });

  const valid = await bcrypt.compare(req.body.password, user.password);
  if (!valid) return res.status(401).json({ message: "Invalid" });

  const token = jwt.sign({ id: user._id }, "secretkey", {
    expiresIn: "1h"
  });

  res.json({ token });
});

👉 Protect routes

const authMiddleware = (req, res, next) => {
  const token = req.headers.authorization;

  const decoded = jwt.verify(token, "secretkey");
  req.user = decoded;

  next();
};

🔐 1.2 Role-Based Access Control (RBAC)
🔥 What is RBAC?

Users have roles, and roles define permissions.

Example:

Role	Permissions
Admin	Full access
Staff	Limited
User	Basic
✅ How to Implement RBAC
Step 1: Add role to user model
const userSchema = new mongoose.Schema({
  email: String,
  password: String,
  role: {
    type: String,
    enum: ["admin", "staff", "user"],
    default: "user"
  }
});
Step 2: Include role in JWT
const token = jwt.sign(
  { id: user._id, role: user.role },
  "secretkey"
);
Step 3: Create Role Middleware
const authorizeRoles = (...roles) => {
  return (req, res, next) => {
    if (!roles.includes(req.user.role)) {
      return res.status(403).json({ message: "Forbidden" });
    }
    next();
  };
};
Step 4: Protect Routes
app.get("/admin", authMiddleware, authorizeRoles("admin"), (req, res) => {
  res.send("Admin only");
});
Step 5: Frontend Role Handling (React)
const role = JSON.parse(atob(token.split(".")[1])).role;

if (role === "admin") {
  // show admin panel
}

1.13 Waste & Disposal Management (Overall Idea)

👉 This module is about:

How your lab safely gets rid of chemicals

You don’t just throw chemicals away — you must:

Handle them safely
Follow environmental laws
Keep records
🧾 1. Log Chemical Disposal
✅ What it means:

Every time a chemical is thrown away, you record it in the system.

🧠 Example:
Chemical: Hydrochloric Acid
Quantity: 500 ml
Disposed by: Ahmed
Date: April 10
Reason: Expired
🎯 Why this is important:
You know what was removed from inventory
You can track hazardous waste
Required for audits and inspections

👉 Without logging, chemicals could be misused or lost.

⚗️ 2. Disposal Methods

Different chemicals need different safe disposal methods. You must record how it was disposed.

🧪 a) Neutralization
✅ What it means:

You chemically make a dangerous substance safe.

🧠 Example:
Acid + Base → becomes neutral (safe)

Example:

Hydrochloric Acid + Sodium Hydroxide → Water + Salt
🎯 Why:
Reduces danger before disposal
Common for acids and bases
🔥 b) Incineration
✅ What it means:

You burn the chemical at high temperature in a special facility.

🧠 Example:
Organic solvents
Contaminated materials
🎯 Why:
Completely destroys harmful substances
Prevents environmental contamination

👉 Must be done in controlled facilities (not normal fire)

📋 3. Regulatory Compliance Logs
✅ What it means:

You keep records to prove:

“We followed the law when disposing chemicals”

🧠 Example:
Disposed according to OSHA/GHS rules
Method: Incineration
Approved by: Safety Officer
🎯 Why:
Governments require this
Needed during inspections
Protects your organization legally

👉 If you don’t have logs → you can face penalties

🌍 4. Environmental Safety Tracking
✅ What it means:

You monitor how disposal affects the environment.

🧠 Example:
Did the chemical contaminate water?
Was it safely neutralized?
Was air pollution controlled?
🎯 Why:
Protect environment 🌱
Prevent pollution
Ensure safe waste handling
🔁 Full Real-Life Flow
Chemical expires or is no longer needed
Lab decides disposal method
Chemical is safely disposed
System records:
What chemical
How it was disposed
Who did it
When
Logs are saved for compliance and audits
🧾 Simple Summary
Log disposal → record every waste action
Neutralization → make chemical safe
Incineration → burn safely
Compliance logs → follow laws
Environmental tracking → protect nature
⚠️ Important Insight

This module is not just technical — it is about:

Safety + Law + Environmental responsibility
